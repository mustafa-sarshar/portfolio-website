SmartGait

The goal of this project is to train multiple Artificial Intelligence (AI) models that can analyze the data gained from Inertial Measurement Units (IMU) to detect/estimate different gait phases (i.e. Foot-Contact, Foot-Off, Mid-Swing) while walking over-ground as well as on the treadmill. The first phase of this project is already completed. Three separate regression models were trained using LSTM algorithm. The results proved the strong potential of these models for above mentioned purposes.

The next phase of the project will get started very soon and the results will be publicly announced.

SQL Fundamentals

When we talk about designing applications for desktop, mobile or web, we simply talk about retrieving data and save it somewhere for later analyses. One of the key elements to save data is to use a data management system that can protect our data and safely perform several complex data manipulations on it.

"The Structured Query Language (SQL) is a domain-specific language used in programming and designed for managing data held in a relational database management system (RDBMS), or for stream processing in a relational data stream management system (RDSMS). It is particularly useful in handling structured data, i.e. data incorporating relations among entities and variables." wikipedia.org

This Repo aims to teach the very important fundamentals of SQL using Postgres, one of the most popular RDBMSs worldwide.
